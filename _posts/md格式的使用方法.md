欢迎来到 **Markdown** 的世界！

如果说 Word 是为打印机设计的（所见即所得），那么 Markdown 就是为互联网设计的（**所写即所想**）。

Markdown 的核心理念是：**不要让排版打断你的写作思路**。你不需要把手移开键盘去点鼠标选“加粗”或“字号”，只需要敲几个符号就能搞定。

这是我为你准备的 **Chirpy 博客专用 Markdown 速查表**：

---

### 1. 最重要的问题：如何换行？

这是 Word 用户转 Markdown 最大的“水土不服”。

*   **分段（大换行）**：
    在 Markdown 里，**敲一次回车不算换行**。你必须**连续敲两次回车**（中间空一行），才会被识别为一个新的段落。
    > **操作**：一段写完，按两下 Enter。

*   **强制换行（小换行）**：
    如果你只想换行但不想分段（比如写诗），需要在**行尾敲两个空格**，然后再按回车。
    > **操作**：`空格` `空格` `Enter`

---

### 2. 标题（Headings）

不要调整字号，而是用 `#` 的数量来表示层级。Chirpy 会自动生成目录。

```markdown
# 这是一级标题 (通常只用于文章标题，正文中少用)
## 这是二级标题 (最常用的章节标题)
### 这是三级标题 (子章节)
#### 这是四级标题
```

---

### 3. 字体格式（Formatting）

Markdown 没有“字体选择”（如宋体/黑体）或“颜色选择”，它强调的是语义。

*   **加粗**：用两个星号包裹。
    *   写法：`**这里要加粗**`
    *   效果：**这里要加粗**
*   **斜体**：用一个星号包裹。
    *   写法：`*这里是斜体*`
    *   效果：*这里是斜体*
*   **删除线**：用两个波浪号包裹。
    *   写法：`~~这句话删掉了~~`
    *   效果：~~这句话删掉了~~
*   **行内代码**（高亮重点）：用反引号（键盘左上角 Esc 下面那个键）包裹。
    *   写法：我们在使用 `git status` 命令。
    *   效果：我们在使用 `git status` 命令。

---

### 4. 插入图片（Images）

在 Chirpy 中插入图片主要有两种方式：

**语法格式**：`![图片的描述文字](图片的路径 URL)`

#### 方式 A：使用网络图片（图床）
如果你有图片的链接，直接填进去：
```markdown
![一只可爱的猫](https://example.com/cat.jpg)
```

#### 方式 B：使用仓库里的图片（推荐）
既然你在浏览器操作，可以这样：
1.  在 GitHub 网页上，把你的图片上传到 `assets/img/` 文件夹下。
2.  在文章里引用它：
```markdown
![我的截图](/assets/img/你的图片名.png)
```
*(注意：路径通常以 `/` 开头，代表根目录)*

---

### 5. 链接（Links）

和图片很像，只是少了一个感叹号 `!`。

*   语法：`[显示在网页上的文字](链接地址)`
*   示例：`欢迎访问 [我的 GitHub](https://github.com)`

---

### 6. 列表（Lists）

*   **无序列表**（圆点）：使用 `-` 或 `*` 加一个空格。
    ```markdown
    - 买菜
    - 做饭
    - 写代码
    ```

*   **有序列表**（数字）：使用数字加点 `.` 再加空格。
    ```markdown
    1. 第一步
    2. 第二步
    3. 第三步
    ```

---

### 7. 引用与提示（Blockquotes）

想要引用名言，或者在 Chirpy 里做一个漂亮的提示框。

*   **普通引用**：使用 `>`
    ```markdown
    > 这是一个引用段落。
    ```

*   **Chirpy 特有的提示块** (Chirpy 专属特效)：
    Chirpy 支持一种很酷的提示框，写法如下：
    ```markdown
    > [!NOTE]
    > 这是一个蓝色的提示信息。
    
    > [!WARNING]
    > 这是一个黄色的警告信息。
    
    > [!DANGER]
    > 这是一个红色的危险警告！
    ```

---

### 8. 代码块（Code Blocks）—— Geek 必备

这是 Markdown 的杀手锏。使用三个反引号 ` ``` ` 包裹一段代码，并指定语言。

**输入：**
\`\`\`python
print("Hello World")
def add(a, b):
    return a + b
\`\`\`

**Chirpy 渲染出的效果：**
会自动带有行号、高亮、以及一个“复制”按钮。
